题目

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例 1：

1
2
3

输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807.

示例 2：

1
2

输入：l1 = [0], l2 = [0]
输出：[0]

示例 3：

2

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]

提示：

每个链表中的节点数在范围 [1, 100] 内 0 <= Node.val <= 9 题目数据保证列表表示的数字不含前导零

type ListNode struct {
    Val  int
    Next *ListNode
}

func addTwoNumbers(l1 *ListNode, l2 *ListNode) *ListNode {
    dummy := &ListNode{} // 定义哑结点
    curr, carry := dummy, 0 // 初始化当前节点以及进位值

    // 循环遍历两个链表，同时将它们的节点值加上进位值
    for l1 != nil || l2 != nil || carry > 0 {
        sum := carry // 记录相加结果的值

        if l1 != nil {
            sum += l1.Val
            l1 = l1.Next
        }

        if l2 != nil {
            sum += l2.Val
            l2 = l2.Next
        }

        // 计算得到相加的结果，并更新进位值
        carry = sum / 10
        curr.Next = &ListNode{
            Val:  sum % 10,
            Next: nil,
        }

        // 将相加结果的个位数作为当前节点值，将进位值作为下一次相加的进位值
        curr = curr.Next
    }

    // 将两个链表中未遍历的节点接到新链表的最后
    if l1 != nil {
        curr.Next = l1
    }

    if l2 != nil {
        curr.Next = l2
    }

    return dummy.Next // 返回哑结点的下一个节点，即为最终的链表
}
