Php Laravel Controller Not Found

2023年3月23日
 | php

问题描述

Route::get('/task', 'TaskController@index');

报错 ： controller [TaskController] does not exist

问题原因

未找到控制器，需要在使用前引入，或只用控制器的相对地址

解决方案：

use App\Http\Controllers\TaskController; # 新增
...
Route::get('task', [TaskController::class, 'index']);
Route::get('task/read/{id}', [TaskController::class, 'read']);

或者

Route::get('/users', 'App\Http\Controllers\TaskController@index');
