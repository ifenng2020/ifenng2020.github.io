Vscode几个快捷键

2023年5月31日
 | tools
  1. 安装Shell command

vscode编辑状态下， cmd + shift + p, 输入 shell command 回车，安装code到PATH

  1. 在终端输入相应指令 打开vscode
$ cd ~/go/src/hello
$ code . # vscode打开当前目录
# 如果使用的是oh-my-zsh，并且打开 vscode plugin
# 也可以实现同样效果
$ vsc
  1. 变量重命名

鼠标移到要修改的关键词上，点击 cmd + shift + L 或 F2 可以实现对单个文件变量的重命名，

另外 光标放到变量上， 输入cmd + d 可选择指定数量变量重命名

效果稍有不同，大家按需使用

  1. 同时编辑多行

Option + 点击 选择要编辑的行

  1. 编辑多行2

选中待编辑文本，shift+option+i,每行末尾出现光标
