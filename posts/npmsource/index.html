Node.js更换国内源

2022年11月5日
 | javascript

更换源很简单，重度Node.js使用者，无法忍受下载插件速度慢，更换源步骤如下

临时使用

1

$ npm --registry https://registry.npm.taobao.org install express

永久使用

1

$ npm config set registry https://registry.npm.taobao.org

验证

1

$ npm config get registry

安装yarn

1

$ npm install -y yarn
