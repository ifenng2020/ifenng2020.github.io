Hugo Maupassant添加favicon.ico

2023年5月15日
 | tools

本篇博文仅在 maupassant 主题下有效 其他主题视情况而定，应该相差不多

生成 favicon

  • 访问 favicon.io
  • 选择 PNG -> ico 或 TEXT -> ico 或 Emoji -> ico
  • 生成完毕点击 Download

添加 favicon 到站点

  • 解压下载文件

  • 将favicon.ico放入 static 文件

$ tree -L 3 ./themes
./themes
└── maupassant
   ├── LICENSE.md
   ├── archetypes
   │   └── default.md
   ├── i18n

   ├── static
   │   ├── css
   │   ├── favicon.ico 	# 这里是新添加的
   │   └── js
   └── theme.toml

放入 static 目录中的 目录 或 文件，都会被 “搬运” 到静态文件目录 public

提交代码

$ git commit -am "add favicon.ico"
